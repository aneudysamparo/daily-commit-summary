# Contributing to daily-commit-summary

Thank you for considering contributing! Please follow these guidelines:

## How to Contribute
- Fork the repository
- Create a new branch for your feature or fix
- Write clear, concise commit messages
- Ensure your code passes linting and tests
- Submit a pull request with a clear description

## Code of Conduct
Be respectful and inclusive. Harassment or discrimination will not be tolerated.

## Issues
- Use issues for bugs, feature requests, or questions
- Provide as much detail as possible

## Pull Requests
- Reference related issues in your PR
- Describe your changes clearly
- One feature/fix per PR

---
Happy coding!
